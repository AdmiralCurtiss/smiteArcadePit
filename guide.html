Arcade Pit Files

There are two zip files below that you need.

The first is an emulator folder with all the files necessary to run Snes9x (1.53) and FCE Ultra 2.2.2. Set them up however you like, including your controller options.

The other file has the roms and states you need to be able to do the challenges. Drop the fcs folder and states folder into the same folder you put your emulators, and add the roms to your own.


  • Dropbox Folder with current files.

To check that your states loaded correctly, try Adventures of the Magic Kingdom on the NES. Press the 1 key to pick the first state and hit 'P' to see if it loads.
For SNES, try Mega Man X. Press F1 this time, because SNES9x uses keys F1-F9 for loading states.

You will need to stream them both emulators to me at 512x448 resolution and a bitrate of 500-1000 kbps. Note that The NES resolution is 512x480, so FCE Ultra will need to be blackboxed a bit. Also make sure MSPaint is ready.
