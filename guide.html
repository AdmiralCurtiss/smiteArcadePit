Arcade Pit Player's Manual

Hello! If you're reading this, it's because you're getting ready to play on Arcade Pit and need to set up your stuff.

You'll need to do the following:

  1. Join the Discord Server
  2. Acquire both of the File Packs containing emulators, states, etc.
  3. Configure OBS to stream directly to me, and double-check the emulator settings (does it load states? Control properly?)
All the information you need is right on this page. There's also a very comprehensive setup guide here. I'm willing to help directly; just ask! None of this is too complicated, despite how it may seem.

1. Discord Channel

Follow this link to join my discord server, for communicating during the show. If you're unfamiliar with Discord, you can use a web version of it or an installed client; both should work fine. There will be some large chat rooms that you can completely ignore if you want, and there will also be a few locked voice rooms, which we will be using. Being on my server will allow me to message you directly in case you need help setting things up and PM you drawing prompts during the show.

2. Arcade Pit Files

You need two zip files, which are both found here: Emulator Pack and Rom/States Pack

The first is an emulator folder with all the files necessary to run Snes9x (1.53), Fusion, and FCE Ultra 2.2.2. Set them up however you like, including your controller options.

The other file has the roms and states you need to be able to do the challenges. Drop everything into the same folder and you should be good to go. If you want to merge it with your own files, just make sure the paths work by trying to load any of the states (batman on the nes has a few you can try).

3. Broadcasting Settings

OBS or XSplit are fine; take your pick and make a new profile/scene collection. You need to set it for a custom streaming server, which is: rtmp://racebox.in/ap and your path/stream key will be a single digit, 1-4, which I will assign. You will need to stream to me at a 512x448 resolution, and a bitrate of 1000 kbps. This is the SNES resolution, so the other two will blackbox a little bit; that's fine! Also make sure MSPaint is ready. Note that Fusion, the genesis emulator, might require monitor capture, or disabling Aero. It is some sort of Windows 7 issue.

I hope to see that you're ready to go by show time (typically 8 pm EST on Sunday). We tend to show up 30m early to make sure everything is ready. If you need more detailed help with the setup, here you go!

Useful Emulator Keyboard Shortcuts

  • Snes 9x: F1-F12 load states 1-12

  • FCEUX (NES): Number keys choose the state and 'p' loads the state

  • Fusion (GEN): Choose the State with F6/F7 and load the state with F8... CAUTION: don't hit F9 (saves over the state)

Show Format

The show will begin with a team battle (a 1v1) of a random game challenge. Whoever wins will control the board, which means they move the single avatar on the map in some direction for a random prize. After their move, the wheel is spun to select what event they play. There are a number of possible events; we will review the rules of each as they appear.

Teams will alternate turns after every event. The only exception is team battles, where whoever wins gains control. The game continues until the 20-point goal trophy is found on the map, at which point a final battle involving all 4 players begins for up to 40 bonus points. Afterwards, the team with the highest score will win. Note: the goal square is randomized on the map, and cannot possibly be found until at least half the map is uncovered.
