Arcade Pit Player's Manual

Hello! If you're reading this, it's because you're getting ready to play on Arcade Pit and need to set up your stuff.

You'll need to do the following:

  1. Join the Discord Server. You should join one of the hangouts about 20 minutes before show time, so we can get ready!
  2. Acquire the challenges pack & bizhawk files the day before the show. I update these files every Friday and you absolutely need the latest ones. (you may need the prereqs files for Bizhawk too).
  3. Configure OBS to stream directly to me, and double-check Bizhawk's settings (does it load states? Controls set properly?)
  4. You'll be using my stream viewer to watch the show, as well as see the other players, with minimal delay.
All the information you need is right on this page. There's also a very comprehensive setup guide here. I'm willing to help directly; just ask! None of this is too complicated, despite how it may seem.

1. Discord Channel

Click https//discord.gg/Smite to join my discord server for communicating during the show. If you're unfamiliar with Discord, you can use a web version of it or an installed client; both should work fine. There will be some large chat rooms that you can completely ignore, and there will also be a few locked voice rooms which we will be using. Once in my server, I can message you directly about setting things up and provide you drawing prompts during the show.

2. Arcade Pit Files

You need a few files, which are found here. You need to get the Challenges Pack, and you need Bizhawk to actually use them.

The Bizhawk prereqs installer will make sure you have C++ and the other assorted stuff that you need to run it (you may already have all of it). Once that's done, the other bizhawk.zip actually contains the emulator. If you are on OSX, you can just grab that one instead.

The challenges pack has everything you need to be able to do the challenges. Drop everything into the same folder as bizhawk. It should ask to overwrite a bunch of folders if you've done it correctly (say yes). To make sure it worked right, load Batman on the NES and press P to try and load a state.

3. Broadcasting Settings

OBS or XSplit are fine; make a new profile/scene collection. You need to set it for a custom streaming server, which is: rtmp://arcadepit.us/ap and your path/stream key will be a single digit, 1-4, which I will assign right before the show. You will need to stream to me at a 512x448 resolution, and a bitrate of 1000 kbps. This is the SNES resolution, so the other two will stretch a little bit; that's fine! Also make sure MSPaint is ready.

I pre-configured Bizhawk to stay at 1024x896 (that's 512x448 doubled). Hopefully it saved this setting. If not, force it to by going to config->Display and setting 'use custom size' to 1024 x 896. Otherwise it will change resolution based on the current console, which is a pain to capture. If the resolution is right, it should be very close to a perfect fit. Click edit scene, then drag the corners of bizhawk outward until both the top/bottom toolbars are hidden on stream.

I hope to see that you're ready to go by show time (typically 8 pm EST on Sunday). We tend to show up 20m early to make sure everything is ready. If you need more detailed help with the setup, here you go!

Useful Emulator Tips

  • F1 - F10 will load stateslots for the current game.

  • You can only edit the controls for a console by opening a game from that console first. Each console has its own controls setup.

  • Ctrl-O chooses the game to open, and it should default to the last folder you were in. So if you are switching between consoles, move up 1 folder.

  • Drag and Dropping games/states works, if you prefer! Find the game you want to play and drop it in bizhawk to make it run.

Show Format

The show will begin with an intro for guests and then we start a team battle (1v1) of a random game challenge. Whoever wins will control the board, which means they move the single avatar on the map in some direction for a random prize. After their move, the wheel is spun to see which event they play. There are a number of possible events; we will review the rules of each as they appear.

Teams alternate turns after every event (the only exception is team battles, where whoever wins gains control). The game continues until the 20-point goal is found on the map, at which point a final battle involving all 4 players begins for up to 40 bonus points. Afterwards, the team with the highest score wins. Note: the goal square is randomized on the map, and cannot possibly be found until at least half the map is uncovered.
