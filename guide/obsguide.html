Connecting and streaming to Arcade Pit

Hey you must've been invited to the show! Cool, but I expect you'll take the time to set things up properly beforehand. this guide will cover every little detail of getting your OBS in top shape for the broadcast. Feel free to ask me questions if you're having trouble.

Note: if you're a savvy OBS user, you won't need this guide. Here is everything you need, summed up: Set your broadcast to "Custom Server" with the FMS URL: rtmp://racebox.in/ap (I will give you the path/stream key when we start). Set your resolution to 512x448, FPS: 60, Bitrate: 1000. No need to stream audio or mic, discord will do that part. Create scenes for all 3 emulators and mspaint (you may have to use Monitor Capture on the Fusion emulator). Make sure you have the most recent files packs.

Everyone will need this open as well, the show viewer: Arcade Pit Viewer. This will let you view everyone in real time. My screen will be the 5th one, mute everyone but me (don't worry, my audio is all set up for no echo).

If you need more details or don't know how to use OBS, the information below has step-by-step instructions.

Step 1: Get the packs and software

  • Arcade Pit Packs - The rom & states pack is usually updated on a weekly basis; make sure you have the latest.

  • OBS - Streaming software. This guide uses the Windows build of OBS 0.657. XSplit or OBS-MP will also work.

Step 2: Setting up the emulators

The emulators are as follows:

  • fceux.exe - NES emulator
  • snes9x.exe - SNES emulator
  • Fusion.exe - Genesis emulator

Set the controls as you see fit for all 3. If you're adjusting the screen size, use a multiplier like 2x or 3x; don't stretch it.

You should also make sure that the roms and states are being found correctly. Open fceu.exe and load 'Batman.' Press 1, to choose state 1, followed by 'P' and it should load the state. If this works, you're in good shape. If not, adjust the folder locations.

Step 3: Configuring OBS Settings

If you already use OBS, then you probably won't want to mess with your existing settings. So the first thing we will do is make a new scene and profile which you can delete later. If you just installed OBS, skip this step.

  1. Open OBS and click 'Profiles' -> 'New' and provide a name.
  2. Next, click 'Scene Collection' -> 'New' and provide a name.

Now we don't have to worry about losing your settings. You can switch back to your old profile and scene collection after the show.

Streaming Settings

Open up your broadcast settings by going to 'Settings' -> 'Settings', and choose 'Encoding' on the left. Set the max bitrate to 1000.

Go down the list on the left to 'Broadcast Settings' where you will be entering the information for my private server. We use my own server because we need to operate with no delay. Change the following things:

  1. For 'Streaming Service' click the drop-down menu and choose 'Custom'. This will let you type in the address.
  2. The FMS URL address is rtmp://racebox.in/ap
  3. Below the FMS URL is another field called Play Path/Stream Key (if any) which we will leave blank for now. I will give it to you before we start (I can't know it in advance; it depends on the teams). It will be a single digit between 1 and 4.

The final setting is on the 'Video' section on the left.

  1. Set your Base Resolution here under custom, to 512 x 448. This is the SNES aspect ratio.
  2. Set your FPS to 30 or 60 (preferably 60).
  3. Ensure you are not using any Resolution Downscale

Step 4: Capture Settings (getting the emulators on OBS)

Hey, we're nearly done now! The complicated part is over. Now we just need to get those emulators on screen. It's up to you if you want them all on one scene or not. The instructions below will have you create 4.

Back in the main section of OBS, you should see two white boxes on the bottom-left. One says 'Scenes' and the other, 'Sources.' You should already have 1 scene. Right-click on the white box and add 3 more. Rename them to: nes, snes, gen, draw.

  • Click the nes scene. The emulator has to be running first, or it won't show up to configure... So with fceu.exe open, right-click on the sources box and add a 'window capture.' Pick FCEUX from the dropdown bar and make sure 'capture mouse cursor' is off.


  • Click the snes scene. with snes9x.exe, you can either do another window capture like before or do a game capture; both should work well but game capture is better. Set it up the same way that FCEU was set up.

  • Click the gen scene. This one is a pain if you're on Windows 7. If you aren't, window capture is fine. For windows 7, you have to either disable Aero or use monitor capture. If you want to use monitor capture, right-click on the sources box and add a 'monitor capture.' Tell it you want to select a sub-region and then click the 'Select Region' button. You'll get a big gray box on your whole monitor. Drag the edges to the insides of your emulator and press enter.

  • For the drawing scene, that's easy. Just open up mspaint (WinKey + R -> 'mspaint.exe') and use window capture. By the way, other drawing programs are against the rules!

Okay, now let's make sure everything looks right. Click the 'Preview Stream' button on the bottom right and whatever scene you're selecting should show up. You can switch them on the fly by selecting them below.

The easiest way to get them to fit the screen properly is to click on the source and press Ctrl-F. It should fit to screen. If not, you can force it to size by clicking 'Edit Scene.' This will let you drag it around directly in the preview window, and resize it too. MSPaint is the one you want only canvas for, so you'll have to do this to get that one to look good.

You don't need to stream audio, so just click on that red mic and red speaker icon to disable them. Ultimately, your OBS should look like this!

Lastly, there is the matter of the show viewer itself: Arcade Pit Viewer. This will let you watch everyone in nearly real-time. My screen will be the 5th one, mute everyone but me (don't worry, my audio is all set up for no echo).

That's everything! This was long-winded but it really comes down to a handful of settings. Again, I can go over anything with you if you need help. Thanks!
